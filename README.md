# AI-Powered Financial Document Intelligence Platform

An AI-powered financial analytics platform that automatically analyzes annual reports and SEC 10-K filings, extracts key financial metrics, compares companies, and generates analyst-style insights.

## Features

- PDF Financial Report Analysis
- Automatic Financial Metric Extraction
- Company Comparison Dashboard
- AI-Generated Analyst Commentary
- Financial Visualization Charts
- CSV Export Functionality

## Tech Stack

- Python
- Streamlit
- Anthropic Claude API
- Pandas
- PDFPlumber

## Example Use Cases

- Financial Analysis
- Equity Research
- FP&A
- Investment Research
- Business Analytics
